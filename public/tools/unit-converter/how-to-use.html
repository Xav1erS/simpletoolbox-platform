How to Use Unitly Converter

Unitly is designed to be intuitive and fast. Whether you're converting a single value or processing a batch of measurements, this guide will walk you through every feature. The entire tool works in your browser—no downloads, no sign‑ups, and complete privacy.

Quick Start: Single Conversion

1 Select a Category

Click one of the category buttons at the top of the converter card: Length, Weight, Temperature, Volume, Area, Speed, or Time. The unit dropdowns will update automatically.

2 Choose Your Units

In the "From Unit" dropdown, select the unit you want to convert from (e.g., "Meters"). In the "To Unit" dropdown, choose the target unit (e.g., "Feet").

3 Enter the Value

Type the number you wish to convert into the Value input box. You can use decimals (e.g., 12.5) and negative numbers. The result will appear instantly as you type—no need to click a button.

4 Use the Result

The converted value is displayed in the Result box. You can:

  • Copy the full result (including units) with the "Copy Result" button.
  • Swap the "From" and "To" units with the "Swap Units" button.
  • Save this unit pair to your favorites for quick access later.
Tip: For temperature conversions, note that Celsius, Fahrenheit, and Kelvin have different zero points, so the formulas are linear rather than multiplicative. Unitly handles this automatically.

Advanced: Batch Conversion Mode

Unitly’s unique batch mode lets you convert multiple values in one go—ideal for homework, engineering calculations, or recipe scaling.

1 Switch to Batch Mode

Click the "Switch to Batch Mode" button above the converter. The interface will change to show a table for entering multiple values.

2 Add Rows

Use the "Add Row" button to create as many input lines as you need. Each row has a value input, a "From" unit dropdown, and a "To" unit dropdown.

3 Fill in Your Data

Enter the values and select the source and target units for each row. You can use the same unit pair for all rows, or mix different conversions.

4 Convert and Export

Click the "Convert" button to process all rows at once. The results appear in a table below, with each row showing the original value, units, and converted result. You can copy individual results or clear the entire batch.

Managing History and Favorites

Unitly keeps your recent conversions handy and lets you save frequently used unit pairs.

  • History: The last 20 conversions are stored locally in your browser. Click the "View History" button to see the list. Each entry shows the value, units, result, and timestamp. Use the "Clear History" button to erase all records.
  • Favorites: While viewing a conversion, click "Add to Favorites" to save that unit pair. Favorites are accessible from a quick‑access panel (coming in a future update).

Theme and Accessibility

Unitly supports both light and dark themes. Click the moon/sun icon in the top‑right corner to toggle. The theme you choose is saved for future visits.

The tool is built with accessibility in mind:

  • Full keyboard navigation (Tab to move, Enter to activate).
  • Screen‑reader‑friendly labels and ARIA attributes.
  • High‑contrast mode compatible.
  • Touch‑friendly buttons (minimal size 44×44 px on mobile).

Offline Usage (PWA)

Unitly is a Progressive Web App. After your first visit, you can install it to your device (via the browser’s "Add to Home Screen" option) and use it without an internet connection. All conversion logic works offline because it runs entirely in your browser.

Tip: To install Unitly as a PWA, look for the install icon in your browser’s address bar or menu. Once installed, it behaves like a native app.
