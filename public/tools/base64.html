Base64 encodes binary data as ASCII text using 64 printable characters (A–Z, a–z, 0–9, +, /). URL-safe mode replaces +→- and /→_, and strips = padding — making it safe for URLs and filenames without percent-encoding. All processing uses the browser's built-in TextEncoder / TextDecoder APIs. Nothing leaves your device.
