How to Use Password Generator

Our free password generator is designed to be intuitive and secure. Whether you need a strong password for a new account or a secure API key for your application, this guide will walk you through every feature.

Quick Start: Generate a Basic Password

1 Choose Password Length

Use the slider or input box to select the desired password length, from 4 to 64 characters. For most accounts, a length of 12‑16 characters is recommended.

2 Select Character Types

Check the boxes for the character sets you want to include:

  • Uppercase Letters (A‑Z)
  • Lowercase Letters (a‑z)
  • Numbers (0‑9)
  • Symbols (!@#$%^&* etc.)

At least one box must be checked. For maximum security, include all four types.

3 Exclude Similar Characters (Optional)

If you want to avoid visually ambiguous characters (like 0/O, 1/l, I/|), check the "Exclude Similar Characters" option. This is useful when you might need to read or type the password manually.

4 Generate and Copy

Click the "Generate Password" button. The new password will appear in the result box. Click the "Copy" button (or the copy icon next to the result) to copy it to your clipboard.

Tip: Each time you generate a password, it is saved in the History panel on the right (desktop) or below (mobile). You can revisit, copy, or delete any previous password.

Advanced: Generate API Keys

Switch to the "API Key" tab to generate secure API keys, tokens, or other long identifiers.

1 Select Key Length

Choose from three standard lengths: 32‑bit, 64‑bit, or 128‑bit. Longer keys provide higher security but are harder to handle.

2 Choose Character Set

Select "Alphanumeric" (letters A‑Z, a‑z, and digits 0‑9) or "Letters Only" (A‑Z, a‑z). Alphanumeric keys are more compact; letters‑only keys avoid confusion with numbers.

3 Generate and Copy

Click "Generate API Key". The key will appear in the result area. Use the copy button to copy it to your clipboard. API keys are also saved in the History panel.

Customizing Advanced Options

Click the "Advanced Options" button (below the main controls) to access additional settings.

  • Custom Prefix/Suffix: Add a fixed prefix or suffix to every generated password (e.g., "app‑" prefix).
  • Exclude Specific Characters: Enter characters you want to exclude from the generated string (e.g., "0O1lI|").
  • Batch Generation: Generate multiple passwords at once (up to 10). This feature is currently disabled but will be enabled in a future update.

Managing Your History

The History panel stores your last 10 generated passwords and API keys locally in your browser. You can:

  • Copy: Click any entry's copy icon to copy that value again.
  • Delete: Click the trash icon to remove a single entry.
  • Clear All: Use the "Clear History" button at the bottom of the panel to delete all saved items.

History is only stored on the device you're using; it will not sync across devices and will be cleared if you clear your browser's local storage.

Theme and Accessibility

The tool supports both light and dark themes. Click the theme toggle (sun/moon icon) in the header to switch. The interface is built with WCAG 2.1 AA standards, ensuring proper contrast and keyboard navigation.

If you encounter any issues or have suggestions, please use the feedback link in the footer. We're constantly improving the tool based on user input.
