Algorithm Museum

Learn about different random number generation algorithms and their applications.

Pseudo Random (Math.random)

Built-in JavaScript algorithm. Fast but predictable if seed known. Good for games and simulations where security isn't critical.

Xorshift128+

Modern algorithm with 2^128-1 period. Very fast and passes most statistical tests. Used in modern browsers for Math.random.

Mersenne Twister

Classic algorithm with extremely long period (2^19937-1). Used in Python, Ruby, and many games. Slow but high quality.

Cryptographically Secure

Uses Web Crypto API for true randomness. Suitable for passwords, tokens, and security applications. Slower but most secure.
