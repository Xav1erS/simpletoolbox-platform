## 设计系统合规性检查清单

### CSS 变量检查
- [ ] 没有定义自定义的 :root 变量
- [ ] 所有颜色使用 --linear-* 变量
- [ ] 所有间距使用 --linear-spacing-* 变量
- [ ] 所有圆角使用 --linear-radius-* 变量
- [ ] 所有阴影使用 --linear-shadow-* 变量
- [ ] 所有过渡使用 --linear-transition-* 变量
- [ ] 所有字体使用 --linear-font-* 变量

### 类名检查
- [ ] 按钮使用 .action-button, .primary-action, .secondary-action
- [ ] 卡片使用标准卡片类名
- [ ] 导航使用标准的 .navigation-area, .nav-container 结构
- [ ] 不使用 .btn, .btn-primary, .btn-outline 等旧类名
- [ ] 不使用自定义的卡片类名（除非特殊需求且符合设计系统）

### 主题切换检查

- [ ] 使用 localStorage.setItem('stb-theme', ...)
- [ ] 主题切换逻辑与其他页面一致
- [ ] html 和 body 都应用主题类名
- [ ] 主题图标使用 Font Awesome 的 sun/moon 图标

### 导航结构检查
- [ ] 品牌标识为 "Simple Toolbox"

- [ ] 导航结构与其他页面完全一致
